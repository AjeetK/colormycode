HIGHLIGHT MY CODE

HIGHTLIGHT MY CODE is a fork of a well known hilite.me project. It is a syntax highlighter which support many programming languages to highlight. The code snippets can be easily highlighted and can be used in emails, blog posts, website and many more.

Copy and paste the source code in left pane, select the language and the color scheme, and click "Highlight!". The readyto use code snippet can be copied from right pane, no external javascript, css are needed.
